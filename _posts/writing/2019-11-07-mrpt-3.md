# 1. MRPT, ROS 에서 사용하기 

## 1. 설치

>  이 포스트에서는 Source level compile 에 대해서는 다루지 않고, ROS에서 사용하기 위한 라이브러리 다운로드 부터 시작한다.
>  Source level compile에 대해서 알고싶다면 [이곳](https://www.mrpt.org/Building_and_Installing_Instructions) 을 참조하면 된다.

우분투에서 bash shell 을 열고, 다음 네가지 방법 중 하나를 수행하여 필요한 라이브러리를 설치한다. CPU 환경, 코드 안정성, 버전 요구사항에따라 적절한 버전을 설치하면 된다. 나는 2번 방법을 활용해서 가장 최신의 2.0 master branch 버전을 설치 했다.

1. **Official repositories** (*최신버전은 아닐 수 있음*)
	```bash
	sudo apt-get install mrpt-apps libmrpt-dev
	```
2. **Git branch: master** (development, mrpt 2.0 branch) **For modern CPUs (SSE3+, ARM v7, AMR v8)**
	```bash
	sudo add-apt-repository ppa:joseluisblancoc/mrpt
	sudo apt-get update
	sudo apt-get install libmrpt-dev mrpt-apps
	```
3. **Git branch: master** (development, mrpt 2.0 branch) **For Older  SEE/SSE2-only CPUs**
	```bash
	sudo add-apt-repository ppa:joseluisblancoc/mrpt-no-sse3
	sudo apt-get update
	sudo apt-get install libmrpt-dev mrpt-apps
	```
4. **Git branch: mrpt-1.5 series** (maintenance branch) **(SSE3+, ARM v7, AMR v8)**
	```bash
	sudo add-apt-repository ppa:joseluisblancoc/mrpt-1.5
	sudo apt-get update
	sudo apt-get install libmrpt-dev mrpt-apps
	```
