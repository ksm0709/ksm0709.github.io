# 준비
MOLA의 공식 (Documentation)[https://docs.mola-slam.org/latest/building.html] 페이지를 참조하면, 다음과 같은 의존성

### Build dependencies[](https://docs.mola-slam.org/latest/building.html#build-dependencies "Permalink to this headline")

-   A C++17 compiler. Either gcc-7, clang-4 or MSVC 2017 are good.
-   CMake >=3.9
-   Mandatory libraries:
    
    -   MRPT >= 1.9.9
    -   GTSAM >= 4.0.2
    -   yaml-cpp (will use an embedded version if not found in the system)
    
-   Optional libraries:
    
    -   opencv
    -   ros1: roscpp, sensor-msgs
