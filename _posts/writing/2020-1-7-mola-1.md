# 준비
MOLA의 공식 (Documentation)[https://docs.mola-slam.org/latest/building.html] 페이지를 참조하면, MOLA는 다음과 같은 의존성을 갖는다고 한다. GCC-7 을 사용하여 Build 해야 함을 알 수 있는데, 이것 때문에 우분투 16.04와 18.04의 설치 방법이 조금 달라진다.

### Build dependencies[](https://docs.mola-slam.org/latest/building.html#build-dependencies "Permalink to this headline")

-   A C++17 compiler. Either gcc-7, clang-4 or MSVC 2017 are good.
-   CMake >=3.9
-   Mandatory libraries:
    
    -   MRPT >= 1.9.9
    -   GTSAM >= 4.0.2
    -   yaml-cpp (will use an embedded version if not found in the system)
    
-   Optional libraries:
    
    -   opencv
    -   ros1: roscpp, sensor-msgs

### C++17 컴파일러 설치 (Ubuntu 16.04 에 해당)
> MRPT 설치시 이미 설치 했다면, 이 항목은 넘어가도 된다.

Ubuntu 18.04 의 경우에는 C++17을 기본으로 지원하지만, 16.04 버전은 그렇지 않다. 따라서, 다음과 같이 수동으로 설치해줘야 한다.

```bash
sudo apt-get install -y software-properties-common
sudo add-apt-repository ppa:ubuntu-toolchain-r/test
sudo apt update
sudo apt install g++-7 -y
```
