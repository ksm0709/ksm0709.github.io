# MRPT 소개
회사에서 모바일 로봇 개발을 할 때, 종종 GitHub을 뒤져서 괜찮은 레퍼런스가 있는지 확인하곤 한다. 그때마다 가끔식 눈에 띄었던 코드가 있었는데, 바로 MRPT [^1] 라는 라이브러리 이다. 그동안 직접 활용 할 일이 없어서 자세히 들여다본적은 없었는데, 이번에 MOLA (A Modular Optimization framework for Localization and mApping) [^2] 이라는 SLAM 프레임워크에 관심을 가지게 되면서, 공부 할 필요성이 생겼다. 왜냐하면 MOLA가 MRPT를 베이스로 사용하고 있기 때문이다. 

여담이지만, MOLA와 MRPT 모두 스페인 University of Almería의 Jose Luis Blanco-Claraco [^3] (줄여서 blanco) 교수님이 리딩하는 프로젝트이다. 그러니 두 프로젝트가 깊은 연관성을 가지는것은 어쩌면 당연한 일이다.

다시 본론으로 돌아와서, MRPT란 **M**obile **R**obot **P**rogramming **T**oolkit 의 약자로, 말 그대로 모바일 로봇을 프로그래밍 하는데에 필요한 기능들을 총 망라 하여 구현 해 놓은 C++ 라이브러리 이다. 왠지 모바일 로봇을 한다면 꼭 배워야만 할 것 같은 느낌을 주는 네이밍이다.

우선, 공부를 시작하기전에 어떤 기능들이 있는지 흝어 봤다. 다음은 대략적으로 파악한 MRPT의 기능들을 나열한 것이다. 대략 흝어봐도 모바일 로봇을 개발하는데 필요한 기능들이 모두 갖추어져 있는 것 같다. (솔직히 하드웨어 드라이버까지 다룬것을 보고 놀랐다)

## MRPT의 기능 리스트

1. **Programming 툴** 
	- Smart Pointer, Threading, Communication(Serial, Socket), File Management(CSerialization), RTTI(RunTime Type Information[^4]), 효율적 예외처리를 위한 매크로 등 프로그래밍 자체를 위한 툴
2. **GUI 툴 및 이미지 프로세싱** 
	- 2D,3D render, 2D Plotting (OpenGL, WxWidget 등을 활용) 
	- OpenCV를 기반으로한 Feature Extration, Rectify, Camera Calibration 등의 기능
3. **수학 도구** 
	- **Linear Algebra** : 로봇 개발에 필수라 할 수 있는 기능인데, eigen3 등의 외부 dependancy를 두지 않고 직접 구현. 기본적인 Matrix, Vector 연산 및 eigen vector, eigen value 추출 등 로봇 개발에 필요한 수준의 기능이 잘 구현되어 있음.
	- **Statistics & Bayes filtering** : Kalman FIlter, Particle Filter, 3D Pose를 표현하기 위한 PDF(Probablility Density Function) 등
	- LM(Levenberg-Marquardt) 알고리즘을 이용한 numeric jacobian estimation
4. **Geometry** 
	- 2D,3D Pose, Transformation등
	- RANSAC, Spline Interpolation, SLERP Interpolation(3D Rotation간의 interpolation 알고리즘), ICP 등이 포함되어 있음
5. **Map** 
	- 다음의 세가지 타입의 맵 형식 지원
		- Occupancy Grid
		- Pose Graph Map
		- HMT(The hierarchical model of metric maps) Map 
6. **센서 추상화 도구** 
	- CObservation이라는 객체로 Lidar, Camera등 활용 가능한 센서 및 Odometry, GPS 데이터 등을 추상화 할 수 있는 도구 지원
7. **SLAM** 
	- 2D Grid SLAM, 3D Pose Graph SLAM 모두 지원
	- Particle Filter 활용한 Localiztion 기능
		- Motion Model(Gaussian, Thrun)
	- 2D
		- RBPF(Rao-Blackwellised particle filtering) SLAM
	- 3D
		- Sparser Relative Bundle Adjustment (SRBA) SLAM
		- HMT(Hybrid Metrical-Topological) SLAM
		- EKF(Extended Kalman Filter) SLAM
		- RO(Range Only) SLAM
8. **Navigation** 
	- **Obstacle Avoidance**
		- VFF(Virtual Force Field)를 활용한 장애물 회피
		- Nearness Diagram(ND)를 활용한 장애물 회피
	- **Planning**
		- value iteration(dijkstra)을 활용한 Simple Planner (PlannerSImple2D)
		- Robot Kinematics를 고려한 TP-RRT( TP Space Rapidly-exploring random tree )
9. **기타** 
	- XML 포맷을 활용한 Runtime 파라메터 저장 및 불러오기 기능
	- rawlog 라는 로봇 데이터셋 녹화, 저장 및 불러오기 기능
	- ini configuration 파일 수정, 저장, 불러오기 기능

---

우선, 가장 밑바닥인 Vector와 Matrix 연산에서 부터시작해서 Mapping뿐 아니라 Navigation까지 모두 커버하는 라이브러리라는 점이 인상깊다. 구현적인 측면도 매우 깔끔해서 괜찮은 라이브러리라는 생각이 든다. 
오히려 ROS에서 SLAM패키지 따로보고, Navigation 패키지 따로보는것 보다 일관성 있게 주행 시스템을 구성 할 수 있을것 같다. 차차 배워가면서 MRPT를 기반으로한 나만의 SLAM, Navigation 시스템을 구축 해 보는것도 좋을 듯 싶다.
